## 个人篇

### 什么是 git

git 是一个用于软件开发的代码 [版本控制系统](https://en.wikipedia.org/wiki/Version_control)。它有以下特点：
* 分布式：git 不依赖于集中的版本控制服务器，不同的用户创建自己的库后，可以自由地在其基础上做版本管理
* 免费开源：就是字面意思

### 理解 git 系统中的各种概念

git 的概念比较多，我解释一下自己认为最重要的几个。

#### commit
commit 是一次代码提交，它也是 git 在切换不同版本时的最小单位。我们平时写代码时，每一次写完一个完整的代码逻辑，就可以做一次代码提交。
